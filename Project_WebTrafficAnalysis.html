CSC115 Final: Web Traffic Analysis

Main About Me Projects UAT Boards Objectives

Project Goal:

Analyze a web traffic dataset to uncover patterns in customer conversions and better understand customer engagement.

Tools Used:

Python

Pandas

Numpy

Matplotlib

Description:

In this project, I analyzed a dataset about web traffic on an ecommerce platform. I extracted data from an external source, prepared the data for analysis, performed EDA on key features, and then applied regression analysis to understand what drives customer conversions. Data visualizations, statistical tests, and regression analysis are all used to answer business questions about customer engagement, exemplifying real-world steps data professionals take to solve problems and communicate their results.
