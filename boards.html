UAT BOARDS OBJECTIVES - DATA SCIENCE

The following is a collection of my strongest personal and academic projects completed during my time as a student at the Universtiy of Advancing Technology.

OBJECTIVE 1


Assess, structure, manage and implement large data sets derived from real-time sources.

End-to-End Data Project: YouTube Metrics Analysis for UAT

In this project, I collect data using YouTube's API, prepare that data, then perform statistical and regression analysis and communicate insights gained.

Web Traffic Analysis

This project involves extracting and analyzing data about web traffic on an ecommerce platform. The data is extracted from an external source, transformed using Python, and analyzed using linear regression to better understand what drives customer conversions and suggest improvements.

OBJECTIVE 2


Compile data from multiple sources, including data selection, data scrubbing and feature engineering, with an emphasis on tidy data and the tidyverse.

Atlas: Your Map for Data Preparation

Atlas is my Student Innovation Project. It is a tool I made that accelerates the process of data preparation by visually highlighting errors in data, making them easy to find and manage - leading to tidier data.

EDA for IoT Traffic

In this project, I import data from an external source, prepare it for analysis, then perform EDA on its features. By transforming the data from its original format through feature engineering, I mirror real-world ETL processes for data integration.

OBJECTIVE 3


Apply statistical tests and tools appropriately to analyze data sets drawn from different types of sources (nature, humankind, organizations, etc.), making inferences and projections from the data.

EDA for IoT Traffic

In this project, I import IoT network data from an external source, prepare it for analysis, then perform EDA on its features. I employ the use of statistical tests and queries to answer potential real-world security questions.

Web Traffic Analysis

This project involves extracting and analyzing data about web traffic on an ecommerce platform. The data is extracted from an external source, prepared for analysis with Pandas, visualized using Matplotlib and Seaborn, and analyzed using a linear regression model from Sci-Kit Learn to better understand what drives customer conversions and suggest improvements.

OBJECTIVE 4


Create visualizations of large data sets in ways that clarify understanding of their meaning and implications.

Visualizing Malicious Traffic

In this project, I extract and load network traffic data from an external source, transforming it for analysis. I then perform statistical analysis to answer security questions and visualize the results.

Mapping Web Data with Spiderfoot

In this project, I scan a publicly available source using Spiderfoot. I then map out and visualize key features in the data to understand and communicate source formats efficiently.

OBJECTIVE 5


Design and implement big data, artificial intelligence and statistical and visual analysis solutions that provide people and organizations with understanding, guidance and options drawn from the data.

Applying Genetic Algorithms in Public Transportation

This project explores the use of genetic algorithms to find optimal bus stop locations in the Maricopa County Area. I load the data from an external source, prepare the data through feature engineering, apply the genetic algorithm, then visualize the results and communicate insights gained.

End-to-End Data Project: YouTube Metrics Analysis for UAT

In this project, I collect real-time data using YouTube's API, prepare that data for analysis through feature engineering, then perform statistical and regression analysis. Finally, I create a report to communicate my results across cross-functional teams.

OBJECTIVE 6


Demonstrate best practices regarding digital privacy and the ethical use of personal information.

Ethical Data Investigation

In this project, I conduct a simulated investigation on a crime scene, applying real-world techniques on digital evidence. I emphasize ethical data usage and create a chain of custody log sheet to ensure data integrity in the investigation.

Student Account Merges

During student account merges, I de-duplicate records within the CampusNexus CRM, employing thorough logging and auditing processes as I go. I ensure PII is handled safely and censor sensitive information in all reporting of the process. This project exemplifies proper use of sensitive personal information and the upholding of user privacy.
