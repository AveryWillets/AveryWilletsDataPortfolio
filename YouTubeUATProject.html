Project Overview

This project explores the use of linear regression analysis to predict YouTube video performance (in views) based on multiple factors. The dataset includes features such as tags used, video length, video title, release date, and more. I utilized Python libraries including Pandas, Matplotlib, and Seaborn to clean, analyze, and visualize the data. I used Scikit-learn for regression analysis and model performance testing.
