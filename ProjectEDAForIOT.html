Project Goal:

Analyze a network traffic dataset to uncover usage patterns and prepare the data for modeling or further exploration.

Tools Used:

Python

Pandas

Numpy

Matplotlib

Description:

In this project, I analyzed a network traffic dataset sourced from Kaggle to explore patterns and prepare it for future analytical use. I uploaded the dataset to Google Colab, parsed the pipe-delimited data into structured columns using custom scripts, and performed cleaning by removing nulls and irrelevant fields. I then conducted exploratory data analysis by answering key questions around traffic volume, protocol types, and source IP activity. This approach mirrors real-world scenarios in data, where clean data is crucial for proper data flow and data-driven decision-making.
