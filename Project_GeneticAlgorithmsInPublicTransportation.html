Applying Genetic Algorithms in Public Transportation

Main About Me Projects UAT Boards Objectives

Project Goals:

Explore the use of genetic algorithms to calculate optimal locations for bus stops in the Maricopa County area.

Tools Used:

Python

DEAP (Distributed Evolutionary Algorithms in Python)

GeoPy

Description:

I begin by identifying and integrating external data from the Valley Metro database. I then prepare the data and perform feature engineering to define population centers. Next, I apply custom area coverage and solution fitness functions. Finally, I apply a DEAP genetic algorithm and search for optimal bus stop locations, then visualize the results. This is just like how genetic algorithms can be used to find optimal hyperparameters for machine learning or to find relevant features in a dataset.
