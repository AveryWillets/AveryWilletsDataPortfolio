Project Goals:

Successfully request a conversion from ZIP code to latitude/longitude via the GeoApify API.

Successfully request restaurant names and information in a given location via the GeoApify API.

Recommend restaurants to the user based on their location and food preference.

Tools Used:

C++

GeoApify

Libcurl

Description:

This project is a restaurant recommender built in C++ that utilizes the GeoApify API to recommend restaurants to a user based on their inputted location and food preference. This project requires many dependencies to work properly, including both static library dependencies and DLL dependencies (depending on device software/hardware). To build this project, ensure to download the attached DLLs into the same folder as the project and to include them in the Linker dependencies for the project. Furthermore, to attain an API key for GeoAPify, simply visit their developer page and copy it from there. The API key should then be added to a config header file and the project should run!
