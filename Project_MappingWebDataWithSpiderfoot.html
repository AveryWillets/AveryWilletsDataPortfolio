Project: Mapping Web Data with Spiderfoot

Main About Me Projects UAT Boards Objectives

Project Goals:

Map data from an unfamiliar source to understand source formats and associations.

Conduct OSINT research on an unfamiliar target and document insights gained.

Tools Used:

Spiderfoot

Python

Description:

In this project, I scan a target website using the Spiderfoot tool. I collect information like associated URLs, connected contacts/contact information, and information captured from form fields. Then, using Spiderfoot, I map out the information gained to understand associations between facts found from the search. This project mirrors a real-world integration scenario where a developer would need to understand their target platform and data formats before developing integration solutions.
