Project Goals:

Explore relevant patterns in malicious traffic found within the dataset and visualize results.

Tools Used:

Python

MatPlotLib

Seaborn

Description:

In this project, I first integrate the malicious network data from an external source. I then prepare the data through feature engineering, ensuring data accuracy along the way. Finally, I address real security questions and visualize the results to efficiently communicate results. I also suggest steps for futher statistical and possibly machine learning analysis.
